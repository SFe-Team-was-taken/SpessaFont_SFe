SpessaFont is a powerful SoundFont2 / DLS Editor that works fully online. It allows to EDIT .sf2 files and .dls files and convert dls to sf2 online for free without any ads! Supports SF2, SF3, DLS, SFOGG, and has full support for converting SF2 and DLS and back! It is written in TypeScript, using React and is licensed under the Apache-2.0 License.
